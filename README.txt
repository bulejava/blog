Установка и запуск блога:

1) Распаковать архив в папку с доменами на локальном вебсервере.
2) Выполнить импорт базы данных из файла в корне проекта dump.sql
3) База данных называется blog, конфигурация подключения к БД настраивается в файле myblog\_\config\database.php
4) Для успешного запуска необходим PHP версии не ниже 5.4
